# scripts/ — Operational scripts

Small utilities that are not part of the web application but are useful during development and demos.

Planned:

- `seed_recipients.py` — load a CSV of recipients into the database
- `run_campaign.py` — trigger a campaign from the command line (alternative to calling the HTTP endpoint)
- `wipe_db.py` — delete the SQLite file and recreate an empty schema (for repeat demos)
